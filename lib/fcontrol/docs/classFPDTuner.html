Class implementing an iso-m tuning algorithm. Once the control frequency specifications are defined, an object of this class can tune (compute controller parameters) of any FPDBlock object based on the specifications and plant model.

Parameters
new_pmFinal system phase margin specification (related to closed loop system overshoot).
new_wcgFinal system crossover frequency specification (related to closed loop system peak time).
new_dtsThe considered loop sampling time.
