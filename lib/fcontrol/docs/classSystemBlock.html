Vector constructor. Creates a transfer fucntion using the coefficients received as numerator and denominator. The positive order of the z varaible is coincident with the vector position, therefore the first position (index 0) defines the coefficient of z^0, second position (index 1) defines the coefficient of z^1, etc...
